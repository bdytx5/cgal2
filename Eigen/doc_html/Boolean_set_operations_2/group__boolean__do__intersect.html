2D Regularized Boolean Set-Operations Reference

Each one of these functions computes if the interior of two given polygons p1 and p2 intersect.

The signature of the function is

bool do_intersect(const Type1 & p1, const Type2 & p2);

Parameters

The types of the paramters of the do_intersect() function are any of the following combinations.
