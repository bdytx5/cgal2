template<class T >
Pointer_property_map<T>::type CGAL::make_property_map ( T *  pointer)

#include <CGAL/property_map.h>

Starting from boost 1.55, the use of raw pointers as property maps has been deprecated.

This function is a shortcut to the recommanded replacement: boost::make_iterator_property_map(<pointer>, boost::typed_identity_property_map<std::size_t>()) Note that the property map is a mutable LvaluePropertyMap with std::size_t as key.
