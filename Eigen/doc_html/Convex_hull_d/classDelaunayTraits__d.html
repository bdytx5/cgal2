Definition

Deprecated:
This package is deprecated since the version 4.6 of CGAL. The package dD Triangulations should be used instead.

Requirements of the first traits class to be used with the class CGAL::Delaunay_d.

Has Models:

CGAL::Cartesian_d<FT,LA>

CGAL::Homogeneous_d<RT,LA>
